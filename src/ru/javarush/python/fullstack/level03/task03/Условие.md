
### Поиск образов в Docker Hub

Найдите все образы, связанные с node, используя команду docker search. Визуально определите, сколько звезд имеет официальный образ Node.js.

Требования:
1. Скрипт должен использовать команду docker search для поиска образов, связанных с node, на Docker Hub. 
2. Визуально определите официальный образ Node.js среди результатов, полученных с помощью команды docker search. 
3. Визуально определите количество звезд, которые имеет официальный образ Node.js.
