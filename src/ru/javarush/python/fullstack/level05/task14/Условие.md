
### Публикация нескольких портов

Запустите контейнер с Nginx и опубликуйте порты для HTTP и HTTPS: 8080 для HTTP (порт 80 в контейнере) и 8443 для HTTPS (порт 443 в контейнере). Веб-сервер должен быть доступен по http://localhost:8080 и https://localhost:8443.

Требования:
•	Контейнер должен быть запущен с использованием образа Nginx, чтобы обеспечить работу веб-сервера.
•	Контейнер должен публиковать порт 8080 хоста, который перенаправляется на порт 80 внутри контейнера для обработки HTTP-запросов.
•	Контейнер должен публиковать порт 8443 хоста, который перенаправляется на порт 443 внутри контейнера для обработки HTTPS-запросов.
•	Веб-сервер должен быть доступен по адресу http://localhost:8080, обеспечивая корректную маршрутизацию HTTP-запросов.
•	Веб-сервер должен быть доступен по адресу https://localhost:8443, обеспечивая корректную маршрутизацию HTTPS-запросов.
