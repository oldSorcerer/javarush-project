
### Масштабирование Nginx

Создайте docker-compose.yml файл для веб-сервера Nginx, который должен быть масштабирован до 3 экземпляров. Каждый экземпляр должен использовать порты из диапазона 80-82 для внешнего доступа.

Требования:
1. Файл docker-compose.yml должен быть создан для описания конфигурации и масштабирования сервисов. 
2. В docker-compose.yml должен быть указан сервис Nginx, который масштабируется до 3 экземпляров. 
3. Каждый экземпляр Nginx должен быть настроен на использование портов из диапазона 80-82 для внешнего доступа. 
4. Docker-compose должен обеспечить, чтобы все экземпляры Nginx были подключены к одной сети для взаимодействия.
