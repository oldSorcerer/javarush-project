
### Передача переменных через команду

Запустите Docker Compose, передав значения переменных NGINX_VERSION и HOST_PORT через командную строку с помощью команды export.

Требования:
1. Переменная NGINX_VERSION должна быть установлена с использованием команды `export` в командной строке перед запуском Docker Compose. 
2. Переменная HOST_PORT должна быть установлена с использованием команды `export` в командной строке перед запуском Docker Compose. 
3. Docker Compose должен быть запущен после установки переменных NGINX_VERSION и HOST_PORT, что позволит использовать их значения в процессе развертывания контейнеров.
